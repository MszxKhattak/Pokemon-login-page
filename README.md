# Pokémon Login Page

A fun and themed **login page for Pokémon lovers**, built using HTML, CSS, and JavaScript. Perfect as a demo or for practice — styled to look like it belongs in the Pokémon universe.

---



## Features

- Pokémon-themed design  
- Responsive layout (works on desktop & mobile)  
- Simple, clean login form  
- Basic form validation (e.g., required fields)  
- Lightweight and easy to customize

---

## Tech Stack

- **HTML** for structure  
- **CSS** for styling  
- **JavaScript** for interactivity
